== Description ==

[Image Viewer Made Easy](http://www.wpexpertdeveloper.com/image-viewer-made-easy/)  is a simple and flexible image viewer/gallery solution with advanced features. You can create image galleries within few minutes while providing advanced features such as rotation, zooming, scaling, flipping and playing as a slideshow. You can us this plugin to create image galleries or provide features such as zooming, scaling, rotating on individual images. 


> **Features**
>
> * Created Unlimited Image Galleries
> * Create awesome Image Viewer(Gallery) with Modal view or Inline view.
> * View images in Full Screen mode.
> * Zoom, Scale, Rotate, Flip images in the viewer.
> * Play the images in the viewer as s slide show.
> * Keyboard support for rotation, zooming, scaling, playing (Modal view).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/image-viewer-made-easy` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin using Plugins section of WordPress.
3. Create new Image Viewer using Add New button of Image Viewer menu item on left menu.
4. Follow the <a href="http://www.wpexpertdeveloper.com/image-viewer-made-easy/">Documentation and Usage</a>.
